# Choose My Car

## Architecture
MVVM with Coordinator (MVVM-C)

## Dependencies
- RxSwift, Swinject and SwiftyMocky Libraries are added using Swift Package Manager.

## Environment
- iOS 13.0 and above
- XCode 12.0
- Swift 5.2
